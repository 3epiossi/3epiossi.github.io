# 物聯網的用途
---
* 工業和製造業 (IIoT)
  * 定義：物聯網領域中增長最快、規模最大的細分市場之一，主要涉及運營技術 (OT) 領域，用於實時監控物理設備的硬體和軟體工具。
  * 特點：需要準實時或實時決策，時延、停機時間和安全性是主要問題。依賴「棕地技術」(brownfield technology)，即老舊硬體和軟體接口。
  * 用例：
    * 對新舊設備進行預防性維護
    * 透過實時需求提高產量
    * 節能
    * 安全系統（熱感應、壓力感應、氣體洩漏）
    * 工廠車間專家系統

* 消費者
  * 定義：最早接入網際網路的細分市場之一，從連網咖啡壺到現在的智慧家庭設備和穿戴設備。
  * 特點：通常是最早採用新技術的市場。標準分歧是一個制約因素，如藍牙、Zigbee、Z-Wave等協議不可互操作。
  * 用例：
    * 智慧灌溉、智慧車庫門、智慧鎖、智慧燈、智慧恆溫器、智慧安全系統
    * 健康和運動追蹤器、智慧服裝/可穿戴設備
    * 寵物定位系統、智慧狗狗門

* 零售、金融和行銷
  * 定義：任何以消費者為基礎的商業交易場所，旨在降低銷售成本和改善客戶體驗。
  * 特點：衡量即時金融交易中的價值，效率和客戶體驗至關重要。
  * 用例：
    * 定向廣告
    * 信標（用於感知客戶、業務模式和間歇時間）作為行銷分析
    * 資產追蹤（庫存控制、損失控制、供應鏈優化）
    * 冷庫監控（分析易腐存貨的冷藏，應用預測分析於食品供應）
    * 資產保險追蹤、駕駛員保險風險測量
    * 商場、酒店或市區的數位標牌
    * 娛樂場所、會議、音樂會、遊樂園和博物館內的信標系統

* 醫療保健
  * 定義：最受關注的領域之一，旨在提高生活品質和降低醫療成本，透過物聯網實現遠程和靈活的患者監控。
  * 特點：嚴格的監管（如HIPAA合規、數據安全）。系統需要像醫療級工具一樣工作，提供全天候、可靠且無停機時間的通信。
  * 用例：
    * 家庭患者護理
    * 預測性和預防性醫療的學習模型
    * 痴呆症患者和老年人護理和追蹤
    * 醫院設備和供應資產追蹤
    * 藥物追蹤和安全
    * 偏遠地區遠程醫療
    * 藥物研究
    * 患者跌倒指示燈

* 運輸業和物流業
  * 定義：使用設備追蹤正在交付、運輸或裝運的資產。也包括車聯網，提供駕駛幫助或預防性維護。
  * 特點：車輛感測器數量增加。重要性延伸到鐵路線和航運船隊，無法承受任何停機時間。系統需要地理定位（GPS導航），分析數據包括資產、時間和空間座標。
  * 用例：
    * 車隊追蹤和位置感知
    * 市政車輛的規劃、路線和監控（除雪、垃圾處理）
    * 冷藏運輸和食品配送安全
    * 軌道車輛識別和追蹤
    * 車隊內的資產和包裹追蹤
    * 道路車輛的預防性維護

* 農業和環境
  * 定義：包括牲畜健康、土地和土壤分析、微觀氣候預測、有效用水等，以及地質和天氣相關災害的預測。旨在提高農業效率，滿足日益增長的糧食需求。
  * 特點：該細分市場在偏遠地區或人口稀疏區域應用會有所不同，影響數據通信系統。
  * 用例：
    * 智慧灌溉和施肥技術，以提高產量
    * 智慧照明在巢式或家禽養殖中的應用，以提高產量
    * 牲畜健康和資產追蹤
    * 透過製造商對遠程農業設備進行預防性維護
    * 基於無人機的土地調查
    * 透過資產追蹤實現農場到市場的供應鏈效率
    * 機器人耕作
    * 用以預防災害的火山和斷層線監測

* 能源產業
  * 定義：包括從生產源頭到消費者的能源監測。大量研究集中在消費能源和商業能源監測器上，如智慧電表。
  * 特點：許多能源生產設施處於偏遠或惡劣環境中。數據可能需要實時或準實時響應，以對能源生產控制系統做出重要反應。
  * 用例：
    * 對石油鑽井平台數千個感測器和數據點進行分析，以提高效率
    * 遠程太陽能電池板監控和維護
    * 核設施的危險分析
    * 在整個城市範圍內部署智慧電表、氣表和水表，以監控使用情況和需求
    * 分時電價定價
    * 根據天氣情況對遠程風力渦輪機進行實時葉片調整

* 智慧城市
  * 定義：「智慧城市」是指互聯和智能化的基礎設施、市民和車輛。是增長最快的細分市場之一，尤其在考慮稅收收入時，成本效益比相當可觀。
  * 特點：傳感器數量龐大，如巴塞隆納部署近100萬個環境感測器。需考慮數量和頻寬的特性。
  * 用例：
    * 透過環境感知進行污染控制和監管分析
    * 利用城市範圍內的感測器網路進行小氣候天氣預報
    * 透過按需廢物管理服務提高效率並改善成本
    * 透過智慧交通燈控制和模式化，改善交通流量和燃油經濟性
    * 按需城市照明提高能源使用效率
    * 根據實時道路需求、天氣狀況和附近的鏟車進行智慧掃雪
    * 根據天氣和當前使用情況對公園和公共場所進行智慧灌溉
    * 智慧攝影機監控犯罪和實時自動AMBER警報
    * 智慧停車場，可根據需要自動找到最佳停車位
    * 橋樑、街道和基礎設施的磨損和使用情況檢測，以提高使用壽命和服務品質

* 軍事和政府
  * 定義：美國市政府、州政府和聯邦政府以及軍方對物聯網的部署都有濃厚興趣。
  * 特點：政府在物聯網中的角色體現在標準化、頻譜分配和法規。
  * 用例：
    * 透過物聯網設備模式分析和信標進行恐怖威脅分析
    * 無人機部署的群體感測器
    * 在戰場上部署感測器炸彈，形成感測器網路，以監測各種威脅
    * 政府資產追蹤系統
    * 實時軍事人員追蹤和定位服務
    * 監測敵對環境的綜合感測器
    * 水位監測，以測量大壩和防洪能力

* 架構師所需了解的系統組件
  * 硬體設計
  * 電源管理和電池設計
  * 嵌入式系統設計和編程
  * 通信系統、無線電信令、協議使用和通信經濟學
  * 網路棧和協議
  * 安全性、服務開通、身份驗證和可信平台
  * 性能分析和系統分區
  * 雲管理、流系統、雲儲存系統和雲經濟學
  * 數據分析、數據管理和數據科學
  * 中間件和設備管理
